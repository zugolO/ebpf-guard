## Use Cases

### Runtime Security Monitoring

Detecting anomalous process behavior, unauthorized file access, and suspicious network connections in Kubernetes workloads.

### Compliance

Monitoring for CIS Kubernetes Benchmark violations and generating audit trails for compliance requirements.

### Threat Hunting

Using behavioral profiles to identify potential compromise indicators that signature-based tools miss.

### Observability

Collecting detailed process and network telemetry for debugging and performance analysis.
